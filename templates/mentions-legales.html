Générateur WireGuard — Application de gestion de tunnels VPN

Identification

Nom de l'application Générateur WireGuard Réalisé par Nciri Wiam Dans le cadre de Un stage en entreprise Type d'application Application web interne de gestion de tunnels VPN WireGuard

Objet de l'application

Le Générateur WireGuard est une application web développée dans le cadre d'un stage. Elle permet de gérer des utilisateurs et de générer des configurations de tunnels VPN basés sur le protocole WireGuard.

L'application est destinée à un usage interne et restreint. L'accès est protégé par un système d'authentification.

Données personnelles

L'application collecte et stocke les données suivantes dans le cadre de son fonctionnement :

— Nom, prénom et entreprise des utilisateurs créés dans l'application.

— Les configurations réseau associées aux tunnels VPN (adresse IP, DNS, endpoint).

Ces données sont stockées dans une base de données MySQL locale et ne sont pas transmises à des tiers.

Conformément au RGPD, vous disposez d'un droit d'accès, de rectification et de suppression de vos données.

Technologies utilisées

Langage Python 3 Framework Flask Base de données MySQL / MariaDB (via XAMPP) Protocole VPN WireGuard (algorithme Curve25519) Hébergement Serveur local interne
